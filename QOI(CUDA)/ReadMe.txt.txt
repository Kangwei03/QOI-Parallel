1. Open QOI.sln

2. Click Project Properties > C/C++ > Language > OpenMP Support (Open the openMP support)

3. Build Solution

4. Open Command Prompt and cd to (drive location)\DSPC\QOI(CUDA)\QOIcuda

5. Enter command "QOI.exe [encode|decode] [input directory] [output directory]"

6. After done running, open (drive location)\DSPC\QOI(CUDA)\QOIcuda, and get the performance csv file

7. Open Google colab or anything IDE with python

8. Open (drive location)\DSPC\QOI(CUDA), find the cuda_graph.txt and copy the code inside to IDE / Google colab

9. Upload the performance csv file into the IDE / Google Colab.

10. Run and get the graph.
